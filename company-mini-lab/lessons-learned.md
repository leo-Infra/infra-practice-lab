# Lessons Learned

## Networking

- Internal network has no DHCP by default
- Machines need manual IP configuration
- Internet access requires a route and correct interface

## Linux

- Netplan is used for network configuration
- Minimal installations do not include all tools
- File permissions are important

## Troubleshooting

- Always debug step by step
- Check:
  - machine state
  - interface
  - IP
  - route
  - service

## Virtualization

- Virtual machines behave like real machines
- Network adapters define connectivity

## Key takeaway

Understanding the "why" is more important than just following steps.
